# 🐶 CODOG (코독)

⭐ GitHub push → 🐾 발자국 생성 → 🎁 아이템 수집 → 🏠 코독 하우스 꾸미기

개발하면 개발자국이 찍히는, 개발자를 위한 셀프칭찬 서비스 **코독(CODOG)** 입니다.  
코딩을 할 때마다 귀여운 발자국이 쌓이며, 개발자에게 작은 성취 경험과 재미를 제공합니다.

---

## 🐶 서비스 소개

개발을 하다 보면 성취감은 크지만, 그 과정을 기록하거나 서로 격려할 공간은 많지 않습니다.

**코독(CODOG)** 은 GitHub 활동을 기반으로 개발자의 코딩 기록을 귀여운 발자국으로 남기고  
다른 개발자들과 응원과 칭찬을 나눌 수 있도록 만든 서비스입니다.

어릴 적 냉장고에 붙어 있던 **‘참 잘했어요’ 도장판**처럼  
코딩을 할 때마다 발자국 도장이 하나씩 찍히고, 이를 모아 나만의 코독 하우스를 꾸밀 수 있습니다.

---

## 🚀 주요 기능

### 🐾 GitHub 활동 연동

- GitHub **push 이벤트 발생 시 발자국 자동 생성**
- 개발 활동을 시각적으로 기록

### 🐾 아이템 시스템

- 모은 발자국으로 **아이템 구매 및 장착**
- 코독 하우스 꾸미기 기능 제공

### 🐾 커뮤니티 기능

- 다른 사용자의 **코독 하우스 방문**
- 응원하기 및 댓글 작성
